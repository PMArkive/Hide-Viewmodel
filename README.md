# Hide Viewmodels
Plugin for toggling weapon viewmodel visibility. Includes cvar for enabling/disabling the feature globally.

## Credits:
- Thanks to [tilgep](https://steamcommunity.com/id/tilgep/) for plugin help

## Commands
Toggle View Model
```
sm_hideviewmodel
sm_hidevm
sm_viewmodel
```
Toggle global feature
```
sm_togglevm
```
